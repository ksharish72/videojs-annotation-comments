# CROSS PLATFORM MOVIE PLAYER WITH ELECTRON JS.

# How to run the app
TO RUN IN DEV:
Switch to videojs folder
1. cd videojs
2.  - Run `yarn install`
	- Run `yarn build`
	- Run `yarn watch`
	- Visit `http://localhost:3004/test.html` to see the magic happen.

To start Electron JS environment
1. npm install
2. npm run electron
